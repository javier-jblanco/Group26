Application Requirements and user interface mock-ups 

Basic ideas:

  • The aim is to show air polution by zones of the city and time of the measurement. 
  • It would allow to check the values in real time

Interface design:

  • "Map" mode: Use an interactive map of the city where the more actualized values are displayed for each zone.
  • "History" mode: for a selected zone (in the map or using a browser), it displays the string of measurements that are available in the database.
