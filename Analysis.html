Analyses performed over the data source:

There are a total of 32247 instances and 45 attributes described as is mention in the following lines:

1) CODI EOI: String code that identifies the measure station.

2) NOM ESTACIÓ: Name of the measure station.

3) ANY: Year of the measure. String value from 2010 to 2021.

4) MES: Numeric month of the measure. String values from 1 to 12.

5) MAGNITUD: Numeric code that identifies the contaminant. String values from 9 to 63.

6) NOM CONTAMINANT: Contaminant chemical formula. 

7) UNITATS: Unit of measure.

8) TIPUS ESTACIÓ: Type of measure station. The possible values are background peri-urban, background-rural, background-urban, industrial-peri-urban, industrial-rural, traffic-peri-urban, traffic-rural, or traffic-urban.

9) CODI INE: String code that identifies the municipality where is situated the measure station.

10) NOM MUNICIPI: Municipality where is situated the measure station.

11-41) DXX (with XX between 01 and 31): Numeric value of the measure for each day of the month.

42) ALTITUD: Numeric altitude of the measure station.

43) LATITUD: Numeric latitude of the measure station.

44) LONGITUD: Numeric length of the measure station.

45) GEOREFERÈNC: Point of a geographic map composed by the latitude and length where the measure station is placed.
